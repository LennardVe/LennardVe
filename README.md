## A Project I'm Particularly Proud Of

For **Mini-Word2Vec-Engine**, I built a word2vec-style word embedding engine completely from scratch in Java including skip-gram with negative sampling, without relying on any existing ML libraries. It was a good excuse to dig deeper into the math behind embeddings, while also showing that something like this can be implemented cleanly without frameworks like TensorFlow or PyTorch.
